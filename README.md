# TutorMate

TutorMate is a web platform for online English tutors to manage students, track lessons, record attendance, generate timesheets, monitor progress, and create AI-supported teaching materials.

## Problem

Many online English tutors use several separate tools such as Google Classroom, WhatsApp, Word documents, and notebooks to manage lessons. This makes it difficult to track attendance, homework, topics covered, student progress, and post-lesson feedback in one place.

Students also do not always check Google Classroom, so tutors often need a faster way to share comments, homework, and learning updates through WhatsApp or email.

## Solution

TutorMate provides one organized dashboard where English tutors can:

- manage student profiles
- record lesson details
- track attendance and homework
- write post-lesson comments
- generate timesheets
- monitor student progress
- create AI-generated lesson plans and speaking materials
- prepare feedback messages for WhatsApp or email

## Target Users

The first version of TutorMate is designed for independent online English tutors who teach students of different ages and levels.

## MVP Features

- Tutor registration and login
- Student profile management
- Lesson tracking
- Attendance tracking
- Homework tracking
- Post-lesson feedback message
- Monthly timesheet generation
- Student progress tracking
- AI lesson assistant for English tutors

## Tech Stack

Planned stack:

- Frontend: Next.js
- Styling: Tailwind CSS
- Authentication and Database: Supabase
- Deployment: Vercel
- AI Integration: OpenAI API or Gemini API
- Reports: PDF export
